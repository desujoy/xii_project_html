Text Formatting
These tags are used for several text formats as following:
<b> - Bold Text
<u> - Underlined Text
<strong> - Important Text
<i> - Italic Text
<em> - Emphasized Text
<mark> - Marked Text
<small> - Smaller Text
<del> - Striked Text
<ins> - Inserted Text
<sub> - Subscript Text
<sup> - Superscript Text
Example
This is Bold
This is Underlined
This is Strong
This is Italic
This is Emphasized
This is Marked
This is Smaller Text
This is Striked Inserted
This is Subscript and Superscript
